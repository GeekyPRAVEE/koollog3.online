Platform

Cloud Platform

The Koollog Cloud hosted data logging platform provides continuous logging and monitoring of temperature, pressure, humidity data and more while giving users instant access from any location. With Koollog Cloud Services, a wireless data logger system can securely transmit data in real-time to be viewed on any internet or data-enabled devices such as a computer, tablet or mobile device.

Scalable Solution with Limitless Applications

With Koollog Cloud Services, overseeing operations at any time from anywhere becomes the new standard. This system is dynamically designed to perfectly fit applications requiring anywhere from a single data logger to networks of hundreds of loggers in multiple locations. Koollog Data Loggers are cloud-ready and can be up and running in just minutes. Unlike many other custom cloud data logging solutions, the Koollog Cloud Services platform offers a production ready “out of the box” solution.

Monitoring and Logging Continuously

The Koollog Cloud offers peace of mind and data assurance. As a monitoring system, the Koollog Cloud allows for instant access to real-time data in the palm of your hand. As a logging solution, recorded data is buffered and saved to the device’s internal memory. This provides uninterrupted continuous monitoring even in the event of a power loss or network failure. When services are restored, all recorded data is transmitted automatically and saved to the file database, providing worry-free around the clock data logging.

Access Data Instantly and Securely from Anywhere in the World

Because all data is saved to the cloud, data can be made available for viewing from any internet enabled location around the world. Global data security can be controlled and configured to allow and restrict access as desired per user. For more restrictive access needs due to regulation concerns, optional in-house network packages are available to contain data access within a facility, campus or intranet.

Email & SMS Event Alert Notifications

The Koollog Cloud platform allows users to configure event alerts to custom fit their needs. Notifications can be sent via email or SMS the moment a threshold is exceeded. Notifications are also available on screen upon login.For any business that requires time sensitive response to temperature, humidity fluctuations, Koollog provides the tools and technology to continuously monitor around the clock with confidence.
